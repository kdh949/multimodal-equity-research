# 백테스트 검증 결과 해석 방법

## 핵심 질문: "내 모델은 유효한가?"

결과를 해석할 때 다음 두 가지를 구분해서 봐야 한다.

1. **예측력 검증** — 모델이 수익률 방향을 맞추는가? (`[1]` `[2]` 섹션)
2. **전략 수익성** — 예측 신호로 실제 수익을 낼 수 있는가? (`[3]` 섹션)

예측력이 있어도 거래 비용을 감안하면 수익성이 없을 수 있고,
예측력이 낮아도 리스크 관리로 인해 포트폴리오 지표가 좋을 수 있다.

---

## [1] Walk-Forward 방향성 정확도

### 판정 기준

| 방향성 정확도 | 판정 | 의미 |
|---|---|---|
| ≥ 55% | **STRONG PASS** | 통계적으로 유의미한 예측력 |
| 50% ~ 55% | **PASS** | 약하지만 실제 예측력 존재. 학술 연구에서도 이 정도면 충분히 유의미 |
| < 50% | **FAIL** | 랜덤 추정보다 나쁨. 모델 재검토 필요 |

### 주의사항

- 방향성 정확도는 fold마다 변동이 있는 것이 **정상**이다. 특정 fold에서 낮아도 전체 평균이 50%를 넘으면 괜찮다.
- fold 수가 5개 미만이면 통계적 신뢰도가 낮다. 분석 기간을 늘리는 것을 권장한다.
- MAE(평균 절대 오차)는 단독으로 해석하기 어렵다. 일일 수익률이 평균 0.1% 수준임을 감안하면 MAE 0.005~0.015 범위는 일반적이다.

### 읽는 법 예시

```
Fold  훈련 시작     훈련 종료     테스트 시작   테스트 종료   MAE      방향성   OOS?
   0  2024-05-04   2024-09-17   2024-09-19   2024-10-17   0.00812  51.3%    -
   1  2024-07-14   2024-11-28   2024-11-29   2024-12-31   0.00744  53.1%    -
  ...
  19  2026-01-05   2026-04-14   2026-04-15   2026-05-04   0.00891  54.7%    YES  ← 가장 최근 OOS

전체 평균 방향성 정확도: 52.4% → PASS
```

OOS(Yes) 행이 실제 미래 예측 성능에 가장 가까운 수치다.

## [2] 종목별 예측 정확도

### 읽는 법

```
종목    예측수   MAE      방향성    평균 예측    평균 실제
NVDA      87   0.00931  57.5%     +0.08%      +0.12%
MSFT      89   0.00712  55.1%     +0.05%      +0.07%
TSLA      84   0.01234  48.8%     +0.03%      +0.09%
```

- **방향성이 높은 종목** (NVDA, MSFT 등): 모델이 잘 예측하는 종목. 이 종목을 중심으로 전략을 구성하면 유리하다.
- **방향성이 낮은 종목** (TSLA 등): 변동성이 크거나 비정형 이벤트가 많은 종목은 예측하기 어렵다.
- **평균 예측 vs 평균 실제**: 부호가 같으면 방향을 잘 맞추고 있다는 뜻이다.

## [3] 포트폴리오 백테스트 지표

### 핵심 지표 해석

| 지표 | 좋음 | 보통 | 나쁨 |
|---|---|---|---|
| CAGR | > SPY CAGR | = SPY 수준 | < 0% |
| 초과 수익률 | > 0% | = 0% | < 0% |
| 샤프 비율 | > 1.0 | 0.5~1.0 | < 0.5 |
| 최대 낙폭 | > -15% | -15%~-25% | < -25% |
| 적중률 | > 55% | 50%~55% | < 50% |

### 지표 간 관계

```
초과 수익률 > 0  →  시장(SPY)보다 나은 전략
샤프 > 1.0      →  리스크 대비 충분한 수익
최대 낙폭 작음  →  극단적 손실 위험 낮음
```

### 실제 출력 예시 해석

```
CAGR (연환산 수익률)       : +18.40%
벤치마크 CAGR (SPY)        : +15.20%
초과 수익률                : +3.20%    ← 시장 대비 3.2% 알파
샤프 비율                  : 1.21      ← 1.0 이상이면 우수
최대 낙폭 (Max Drawdown)   : -12.30%   ← 15% 미만이면 양호
적중률 (Hit Rate)          : 54.60%
```

이 결과라면: 모델이 시장 대비 초과 수익을 내고, 리스크도 적절하게 관리되고 있다.

## [4] OOS 예측 vs 실제 샘플

### 읽는 법

```
날짜          종목    예측 수익률   실제 수익률    오차       방향 맞춤?
2026-04-15    AAPL    +0.21%        +0.43%       0.22%       O
2026-04-15    MSFT    +0.15%        -0.08%       0.23%       X
2026-04-16    NVDA    -0.10%        -0.23%       0.13%       O
```

- `O`: 상승/하락 방향을 맞춤
- `X`: 방향을 틀림
- 예측 수익률의 절대값보다 **부호(방향)**가 더 중요하다

## 결과가 나쁠 때 점검 체크리스트

| 증상 | 점검 항목 |
|---|---|
| 방향성 정확도 < 50% | 피처 누수(look-ahead bias) 여부 확인 |
| Sharpe < 0.5 | 거래 비용이 너무 높거나 신호 빈도가 낮음 |
| 초과 수익률 음수 | 종목 선정 기준 재검토 |
| fold마다 정확도 편차 큼 | 훈련 기간 연장 (`train_periods` 증가) |
| 최대 낙폭 > -30% | 포지션 사이즈 또는 손절 기준 재검토 |

## 다음 단계

1. 방향성 정확도 PASS → FinBERT, Chronos 등 고도화 모델로 재검증
2. 초과 수익률 양수 → 실제 포지션 사이즈 최적화 진행
3. 특정 종목 방향성 낮음 → 해당 종목 제외 또는 가중치 축소 검토
